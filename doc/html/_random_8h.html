Detailed Description

Controls the fate of the player.

This file contains all free functions which are used to generate random numbers. It includes standard random number generation, dice rolls, and weighted random number generation. It also allows for the generation of random numbers contained in various distributions.

Simulates several throws of a die and returns the sum of results.

This function uses the Dungeons & Dragons style rolling notation. It rolls an x-sided die n times, and returns the combined sum. The result is a reasonably normal distribution with a maximum of n * x and minimum of x.
